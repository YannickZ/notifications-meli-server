## Introducción
 - Para instalar y correr este proyecto, es necesario tener Node.js y una instancia de mongodb instalada y corriendo.

## Tecnologías
 - Node.js
 - Express
 - MongoDB

## Instalación de dependencias

```sh
$ npm i
$ node_modules/.bin/bower install
```

## Ejecución
```sh
$ npm run dev
```
